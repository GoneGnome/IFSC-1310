Progressive Enhancement

Progressive Enhancement is a method of web design that focuses on planning/creating your project using the most basic of web design techniques. Since your foundation is built on strong simple functionality, there is always a strong foundation to come back upon if a specific iteration doesn't work out well.

Progressive Enhancement is the opposite of Graceful Degradation in which you build from simplicity to complexity instead of working vice versa. It allows for easier troubleshooting and redesign since, the core functionality and content is built in a strong simple base.
